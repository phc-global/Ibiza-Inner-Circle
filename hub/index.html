How to start

  1. Tap Open your cloud machine and wait a moment for it to build.
  2. In the terminal, run claude and sign in with your Claude account.
  3. Once in, run claude --dangerously-skip-permissions to let it build freely.
  4. Paste a prompt below and watch it build the real thing.
